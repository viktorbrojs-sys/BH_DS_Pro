# HelpDeskNeuro — PyTorch чат-бот для Telegram

## Описание
Telegram-бот, который определяет IT-проблему пользователя с помощью нейронной сети
и направляет к нужному специалисту: Программисту, Системному администратору или Электронику.

## Стек
- Python 3.11
- PyTorch 2.x (FeedForwardNet — 2 скрытых слоя + BatchNorm + Dropout)
- NLTK (токенизация, стемминг, bag-of-words)
- aiogram 3.x (Telegram Bot API)
- Flask (статус-страница на порту 5000)

## Структура файлов
```
intents.json       — Намерения (теги, паттерны, ответы) — 22 темы
nltk_utils.py      — Токенизация, стемминг, bag-of-words
model.py           — Класс FeedForwardNet (PyTorch)
train.py           — Обучение с логированием в logs/training.log
chat.py            — Загрузка модели, предсказание, логирование Q&A
main.py            — Telegram-бот (aiogram) + Flask статус-сервер
requirements.txt   — Зависимости Python
trained_data.pth   — Обученная модель (генерируется при первом запуске)
logs/
  training.log     — Лог обучения: эпохи, loss, accuracy
  chat.log         — Лог вопросов/ответов: user_id, tag, conf, Q, A
  bot.log          — Общий лог бота
```

## Архитектура нейросети

**Тип: Feed-Forward Network (FNN)**

Выбор обоснован задачей: классификация намерений по вектору bag-of-words —
фиксированный вход, конечное число классов. Рекуррентные сети (LSTM/GRU) здесь избыточны.

```
Вход (bag-of-words) -> Linear -> BatchNorm -> ReLU -> Dropout(0.3)
                    -> Linear -> BatchNorm -> ReLU -> Dropout(0.3)
                    -> Linear (логиты по числу намерений)
```

- **Loss**: CrossEntropyLoss
- **Optimizer**: Adam
- **Scheduler**: StepLR (×0.5 каждые 200 эпох)

## Команды Telegram
- `/start` — приветствие
- `/help`  — справка по возможностям
- `/status` — статус модели и счётчик обработанных сообщений

## Запуск

1. Установить зависимости:
```bash
pip install -r requirements.txt
```

2. Задать токен бота:
```bash
export TELEGRAM_BOT_TOKEN=ваш_токен_здесь
```

3. Запустить:
```bash
python main.py
```

При первом запуске модель обучается автоматически (~500 эпох, занимает несколько минут).
После обучения бот запускается и начинает отвечать.

Статус-страница: http://localhost:5000
